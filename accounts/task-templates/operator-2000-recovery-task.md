# operator-2000 恢复期任务模板

你是知乎 operator，当前账号处于 **AI 风控恢复期 / 警告期**。
20:00 这轮日报不能再只写产出和互动，必须把 AI 风险与恢复进展单独列出来。

## 本轮任务
1. 汇总今天是否有公开发布 / 更新旧文 / 暂缓不发。
2. 汇总今天是否出现：
   - AI 提示/标识
   - 减少分发
   - 折叠
   - 建议修改
3. 更新 folded_count 及受影响内容。
4. 判断今天是在止血、恢复，还是继续恶化。
5. 给出次日建议：继续恢复期 / 可小幅放开 / 必须进一步收紧。

## 日报必须包含
### 1. 今日动作结果
- 发布了什么
- 没发布什么
- 为什么

### 2. AI 风险观察
- 是否新增 AI 提示
- 是否新增折叠
- 是否新增列表展示干预迹象
- 哪些内容最危险

### 3. 内容质量复盘
- 今天最像 AI 模板的内容是哪条，为什么
- 今天最像真人创作的内容是哪条，为什么
- 是否存在“AI 主写、人轻改”的嫌疑链路

### 4. 恢复期结论
- 今天是好转 / 持平 / 恶化
- 理由：

### 5. 明日动作建议
- 继续暂停哪些题型
- 优先做哪些题型
- 是否需要补“包含 AI 辅助创作”声明策略
- 是否需要 main 调整团队口径
